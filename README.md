## Overview

This repository contains an ETL (Extract, Transform, Load) pipeline developed in Python that consumes data from the public DummyJSON API, using the following endpoints:

- https://dummyjson.com/users  
- https://dummyjson.com/products  

The pipeline extracts, cleans, transforms, and stores the data in Parquet format, making it ready for analysis.

This project was developed as part of an academic practical assignment.

---

## Objectives

- Build a complete ETL/ELT pipeline using Python  
- Consume data from a public API  
- Select relevant fields  
- Perform data cleaning and transformation  
- Generate Parquet files  
- Produce analytics-ready datasets
